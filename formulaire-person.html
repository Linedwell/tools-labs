Création d'ébauche de mangaka

Cette page est un formulaire qui permet de générer le code wiki d'une ébauche de mangaka. Il faut premièrement compléter le formulaire puis valider avec le bouton « générer le modèle » en bas. Ce bouton peut être utilisé plusieurs fois consécutivement lorsque par exemple un champ a été modifié. Le code wiki généré n'a ensuite plus qu'à être copié-collé sur Wikipédia dans la zone d'édition de l'article à créer.
Version 1.4 : le code reste en perpétuelle évolution pour refléter les changements des articles et des catégories présentes sur Wikipédia. N'hésitez pas à passer sur la page de discussion pour demander des corrections, des modifications ou simplement des explications.
Le formulaire ne fonctionne pas avec Microsoft Internet Explorer, merci d'utiliser un autre navigateur.

Classement :
Permet de personnaliser le classement de l'article dans les catégories. Voir l'aide complète concernant les catégories. Le bouton « générer ce champ » permet de réaliser les traitements classiques, comme retirer les caractères accentués ou supprimer un article défini placé en début de titre.
Section : Permet de rajouter dans le code wiki des sections vides. Ces titres sont là pour créer un découpage de l'article homogénéisé. Aucun ne devrait être généré si du texte n'y est pas ajouté lors de la création de l'ébauche.
Note : Les selections multiples se font à l'aide de CTRL+clic.
